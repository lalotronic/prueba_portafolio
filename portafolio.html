Primer proyecto de curriculum

Este proyecto presenta un curriculum vitae básico desarrollado utilizando HTML y CSS. Ofrece una estructura simple y fácil de leer, resaltando la información personal y profesional de manera clara y concisa. Es una introducción ideal al desarrollo web para aquellos que están comenzando en el campo de la creación de páginas web.

Land page Iguana

Este proyecto muestra postura en que puedes organizar tus fotos para una landing page, esta orientado a contenido de publicidad ideal para tener material de difusión en tus plataformas online de ventas o para tu sitio web. Es una carta de presentación para inspirar a tus clientes en la exploración y descubrimientos de servicios o productos que ofreces.

Proyecto Cupon

Este es un ejemplo del uso de cards en bootstrap para generar descripciones gráficas de algún producto que deseas promocionar. Este ejemplo muestra las ofertas de cupones con productos o sevicios con descuentos.
